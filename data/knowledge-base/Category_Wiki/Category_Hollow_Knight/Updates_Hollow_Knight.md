# Updates (Hollow Knight)

Source: https://hollowknight.fandom.com/wiki/Updates_(Hollow_Knight)

Software updates are written in a 4-decimal format notation:

MAJOR.MINOR.MAINTENANCE.BUILD

First and second decimals are for major/minor feature additions, respectively. The third decimal is for general non-content-adding changes. The fourth is for fixes only - no changes (apart from obvious changes that are part of a bugfix).
