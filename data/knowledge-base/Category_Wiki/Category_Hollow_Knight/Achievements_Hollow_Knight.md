# Achievements (Hollow Knight)

Source: https://hollowknight.fandom.com/wiki/Achievements_(Hollow_Knight)

Base Game Achievements (52)

Charm Achievements

There are a total of 45 Charms to be found in the base game, from which the Knight can collect only 40 at a time. Please refer to the Charms page for more information.

Mask Shards

There are a total of 16 shards to be found in the game. Please refer to the Mask Shard page for a list of locations.

Vessel Fragments

There are a total of 9 fragments to be found in the game. Please refer to Vessel Fragments page for list of locations.

Bosses

Dream Essence

There is a total of 3206 Essence that can be acquired, not counting Essence occasionally dropped by enemies. Please refer to Dream Nail for a list of Essence locations.

Grubs

There are a total of 46 Grubs to be found in the game. Please refer to Grubs for a list of Grub locations.

Stag Stations

There are a total of 11 Stag Stations to be unlocked in the game. Please refer Stag Stations page for a list of locations and toll prices.

Miscellaneous

The following Achievements are related to NPCs and their quests. Please see their individual pages for more information.

Challenges

Endings

There are five endings to Hollow Knight , four of which involve completing an optional quest to unlock, with two of these endings being added by Godmaster (the related achievement can be found in the Godmaster section of this page). For more information, please see Endings .
