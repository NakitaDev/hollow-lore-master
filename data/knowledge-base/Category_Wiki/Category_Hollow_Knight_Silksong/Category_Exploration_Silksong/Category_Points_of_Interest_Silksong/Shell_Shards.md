# Shell Shards

Source: https://hollowknight.fandom.com/wiki/Shell_Shards

Shell Shards are a collectible in Hollow Knight: Silksong .

Use

Shell Shards are used to repair and craft the Tools that Hornet uses alongside her Needle . Tools can only be repaired when resting at a Bench and different Tools require a different number of Shell Shards to be repaired.

How to Acquire

Shell Shards can be gained in several different ways:

Defeating enemies Destroying Shell Shard deposits Breaking down Shard Bundles
